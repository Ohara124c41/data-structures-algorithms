# Basic Algorithms

Seven practice problems with Python solutions and brief explanations.

## Files
- `problem_1.py` … `problem_7.py`: solutions with inline tests and expected-output comments.
- `explanation_1.md` … `explanation_7.md`: design/efficiency notes.
- `Trie.ipynb`: filled notebook for the trie/suffixes exercise.

## Running
From this folder:
```bash
python problem_1.py
python problem_2.py
python problem_3.py
python problem_4.py
python problem_5.py
python problem_6.py
python problem_7.py
```
Each script prints its test outputs.

## Notes
- Standard library only; no extra dependencies.
- Sorting in problem_3 uses a custom merge sort (no built-ins).
