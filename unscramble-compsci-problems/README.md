# Unscramble Computer Science Problems

Five small tasks on call/text records for September 2016. Code is split across Task0–Task4; runtime analysis in `Analysis.md`.

## Files
- `Task0.py` … `Task4.py`: solutions with required outputs (and expected-output comments).
- `calls.csv`, `texts.csv`: input data.
- `Analysis.md`: Big-O notes for each task.

## Running
From this folder:
```bash
python Task0.py
python Task1.py
python Task2.py
python Task3.py
python Task4.py
```
Each script prints the required answer lines.

## Notes
- No extra dependencies beyond Python stdlib.
- Task3/4 sorting uses built-in `sorted` as allowed.
